# Student Performance & Dropout Prediction Dashboard

## Project Overview
Developed an interactive Power BI dashboard using student marks, attendance, and assignment completion datasets to analyse academic performance and dropout risk.

## Tools Used
- SQL
- Power BI
- Excel/CSV

## Key Features
- Attendance vs Marks Correlation Analysis
- Subject-wise Performance Tracking
- At-Risk Student Identification
- Interactive Dashboard Visualizations
- Data-Driven Insights for Student Retention

## Dashboard Insights
- Students with low attendance tend to have lower academic performance.
- Subject-wise analysis helps identify weak areas.
- At-risk students can be identified early for intervention.
